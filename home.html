Linux system information

The following modules are available

cpuinfo

Displays the system's processor information as reported by the proc filesystem.

This module executes the 'cat' command and uses a socket stream to send the output of /proc/cpuinfo to the browser.

diskfree

Displays the disk usage and free space of all drive partitions currently connected to the system.

This module executes the 'df' command with '-h' option and uses a socket stream to send output to the browser.

issue

Displays the current system's issue: the distribution, os/kernel, and version codename.

This module copies the file /etc/issue.net to the socket descriptor.

processes

Displays all processes currently running on the system by process ID and reports the executing user and group ownership.

This module dynamically generates the process list by reading the /proc directory and extracting the information from each process's directory.

time

Displays the system time.

This module uses the 'gettimeofday' function to get system time and formats it for printing with the 'strftime' function.

top

Displays various process and memory information on the running system.

This module executes the 'top' command and immediately exits with the'-bn1' option.
